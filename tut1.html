So today I'll be teaching you how to make a web page like the one I have here! It's way way simple and can be done by anyone. Just follow these simple steps:

Download and install an IDE(Integrated Development Environment). My favourite and most popular is Visual Studio Code This is merely just software that will allow you to create and edit HTML pages amongst other uses.

Since this is a beginner tutorial, it'll be much easier to edit an already made template. Download a html template thats most appealing to you. This site has a bunch of free cool ones.

The templates usually come in zipped files. Usually your computer comes with 7zip for unzipping. If not: find it here. Winrar is another common one.

In your Zip file, locate the index.html file. It is usually the main page of the website. Open it with your browser and as well as with the downloaded Visual Studio Code(VSC). Using VSC change the white text into your desirable text and save. Refresh the index page on the browser and your change should be reflected.
